@cloud-materials/charts-common 图表预览

根据包内 artifacts 文档整理，核心 CChart 类型包含 line、bar、histogram、pie、mix；另外还提供 CloudEChart 原生 ECharts 封装、BigData 示例，以及 CVolcMap/CByteplusMap 地图能力。

line折线 / 平滑 / 阶梯
bar柱状 / 堆叠 / 横向条形
histogram直方 / 柱形变体
pie饼图 / 玫瑰图
mix折线 + 柱形混合
map火山 / BytePlus 地图

折线图

type: line

适合趋势分析，支持多指标、平滑曲线、阶梯线和序列聚焦。

柱状图

type: bar

适合分类对比，支持多指标、标签格式化和堆叠。

横向条形图

bar option

适合排行类数据，BigData 示例里也包含横向大数据柱形方案。

堆叠柱状图

stack

用于展示总量和组成结构，文档里的 bar/histogram 都有 stack 示例。

直方/柱形图

type: histogram

charts-common 中的 histogram 是柱形变体，支持多指标、堆叠和最小柱高等行为配置。

饼图

type: pie

适合占比结构分析，支持 tooltip、legend 和标准样式。

南丁格尔玫瑰图

pie rose

文档中通过 behavior.useRose = "radius" 启用玫瑰图效果。

折柱混合图

type: mix

支持左/右 Y 轴，适合同时展示数量和比例等不同量纲指标。

大数据折线图

CloudEChart

BigData 文档中演示了 large 模式，这里用 5000 个点做轻量预览。

地图图表

CVolcMap / CByteplusMap

包内包含火山地图和 BytePlus 地图文档，实际渲染通常需要地图 SDK、密钥或业务地图数据。

地图能力已包含 可用于区域、点位、轨迹等地图场景；需要接入对应地图配置后展示。
本页面为静态预览，使用本地 node_modules/echarts/dist/echarts.min.js 渲染；组件 API 以 @cloud-materials/charts-common 文档为准。
